# Sales Analytics Dashboard

## Project Overview
This is a beginner Power BI dashboard project created using Excel sales data.

## Tools Used
- Power BI
- Excel
- DAX

## Dashboard Features
- KPI Cards
- Sales by Product
- Sales by Region
- Sales Trend Over Time
- Interactive Slicer Filters
